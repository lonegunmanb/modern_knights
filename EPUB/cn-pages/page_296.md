
Mercier 是 Paul Vankerkhoven 的一位熟识。 他也是 Ligue Internationale de la Liberte（国际自由联盟，世界反共联盟在比利时的分支）成员。 此外，他与 Otto van Habsburg 的私人秘书 Jacques Jonet 等人一起，坐在那个极其阴暗的“Société Internationale de Wilton Park”（Wilton Park 国际协会）中，那里也出现一位 Jean Violet（来自“Cercle Violet”）。 Mercier 与 Ordre du Rouvre（Ordre du Rouvre 骑士团）骑士之间还有其他联系 7.

在相关的国家安全（Sûreté de l'Etat）备忘录中，Bernard Mercier 也被提及为“将一名在逃的新纳粹恐怖分子从安特卫普转移到布鲁塞尔 CEPIC 总部的组织者之一”。 据称该恐怖分子必须由 Mercier 转移到位于阿登、属于 Bonvoisin 家族的一处房产里。 这名德国人从安特卫普赴布鲁塞尔途中被国家安全局拦截。 Mercier 予以否认。 《De Morgen》写道此人系在逃的新纳粹恐怖分子 Eckerhard Weil，并且“法院里应该有一个卷宗”。 8

在记者 René Haquin 的著作《Operation State Security - State Security - State Security - the State Security and the WNP》（暂译：行动国家安全——国家安全与 WNP）中，Michel Libert 获得发言权。 Libert 讲述：“Paul Latinus 的确是 Westland New Post 的领导人，但在他之上还有其他与国外保持联系的人。” 其中提到了 Bernard Mercier 9。 Haquin 随后断言 Bernard Mercier 几乎在所有极右翼重大丑闻中持续出现，尤其是在 WNP 档案中 10。 记者还公布了一份 1983 年 5 月比利时国家安全局内部报告的结果。 据该报告（Haquin 所写），Mercier 是驻科隆的 Teutonic Order（条顿骑士团）一位总长名下 WNP 的“Landers vertreken Inspektor” 11。 这是那个曾经或现在在比利时由 Karl De Lombaerde 领导的骑士团…… 奇怪…… Mercier 在所有这些事件中的角色仍然非常模糊。 或许下文可解释一些情况：Mercier 主要是 Ordre de la Milice de Jesus-Christ（耶稣基督民兵骑士团）的 Devotion 骑士，该骑士团当时与 Ordre du Rouvre 以及 Milice Sanctae Mariae（圣母民兵团）签署了合作协议。 该协议意味着右翼力量在其为维护社会中的基督教价值进行斗争中实现了聚合。

“通过宣讲来关怀灵魂的得救”。 这是西班牙神父 Dominicus（1170-1221）心中的愿景。 很快 Dominicus 赢得支持。 修会成员、平信徒与司铎加入了 Ordo fratrum praedicatorum（传道者兄弟会修会），即“传道者修会”，也就是今日所称的多米尼加会（Dominicans）。
